## 1. Первый запуск

### Что нужно чтобы всё заработало:
1. Добавить хотя бы один Telegram аккаунт
2. Прогреть его 5 дней
3. Добавить каналы-источники
4. Создать оффер
5. Запустить рассылку

## 2. Аккаунты `/accounts`

Это Telegram аккаунты с которых идут рассылки.

### Добавить через Генератор Session (рекомендую)
1. Зайди на `/accounts/session-gen`
2. Введи номер телефона с кодом страны: `+79991234567`
3. Нажми **"Получить код"**
4. Открой Telegram на этом номере — придёт сообщение с кодом
5. Введи код на сайте → нажми **"Подтвердить"**
6. Аккаунт автоматически добавится в систему

### Добавить через .session файл
1. Нажми **"+ Аккаунт"** → введи номер телефона
2. Рядом с аккаунтом нажми кнопку **"Session"** → загрузи .session файл
3. Нажми иконку Wi-Fi → аккаунт подключится

### Настройки аккаунта
- **Лимит/день** — сколько сообщений в день отправляет этот аккаунт (рекомендую 30-50)
- **Задержка мин/макс** — пауза между сообщениями в секундах (рекомендую 20-60 сек)

### Прокси
Рекомендуется для каждого аккаунта. Нажми **"+ Прокси"**:
- Тип: SOCKS5 или HTTP
- Host, Port, Login, Password
- Страна (например RU, NL, DE)

## 3. Каналы `/channels` и Поиск `/discover`

### Добавить канал вручную
1. Каналы → **"+ Добавить"**
2. Введи username: `@cryptowhales` или просто `cryptowhales`
3. Выбери категорию (крипта, гемблинг и т.д.)
4. Нажми добавить

### Найти каналы автоматически
1. Поиск → выбери аккаунт
2. Нажми на нишу (Крипта, Гемблинг, Арбитраж, Финансы, Нутра)
   **или** введи свой запрос и нажми поиск
3. Нажми **"+"** рядом с понравившимся каналом

### Парсинг участников
Это сбор базы пользователей из канала для рассылки.
1. В списке каналов нажми иконку **↓** рядом с каналом
2. Система запустит парсинг (нужен активный аккаунт)
3. Пользователи сохранятся в базу и будут доступны для рассылки

## 4. Рассылки `/campaigns`

### Создать рассылку
1. Нажми **"+ Новая рассылка"**
2. Заполни:
   - **Название** — для себя
   - **Тип**: Личные сообщения / Комментарии / Инвайтинг / Постинг
   - **Текст сообщения** — что будет отправляться
   - **Каналы-источники** — откуда берутся получатели
   - **Аккаунт** — с какого аккаунта слать (или Авто)
   - **Задержка** — пауза между сообщениями
3. Нажми **"Создать рассылку"**

### Запустить рассылку
1. В списке рассылок нажми **"Запустить"** рядом с нужной
2. Статус изменится на "Идёт"
3. Следи за прогрессом — видно сколько отправлено/доставлено/ошибок

### Советы
- Не ставь задержку меньше 20 секунд — аккаунт получит бан
- Лучше использовать несколько аккаунтов с ротацией
- Если аккаунт получит FloodWait — система автоматически переключится на следующий

## 5. Офферы `/offers`

### Добавить свой оффер
1. Нажми **"+ Свой оффер"**
2. Заполни название, URL, выплату за лид

### Подключить CPA-сети
Admitad, LeadGid, Alfaleads — получи API ключи на их сайтах и добавь в Railway Variables:
- `ADMITAD_TOKEN`
- `LEADGID_API_KEY`
- `ALFALEADS_API_KEY`

После добавления нажми **"Синхронизировать"** — офферы подтянутся автоматически.

### Создать UTM-ссылку
Нажми **"UTM"** рядом с оффером → получишь короткую ссылку для отслеживания кликов.

## 6. Шпион конкурентов `/spy`

Показывает рекламные посты в каналах.

### Как использовать
1. Выбери аккаунт для сканирования
2. Выбери канал из своего списка
3. Нажми **"Сканировать канал"**
4. Система проанализирует последние 100 постов и найдёт рекламные

### Что показывает
- Текст рекламного поста
- Ссылка на оффер
- Рекламодатель (домен)
- Количество просмотров
- Дата публикации

### Кнопка "В офферы"
Нажми рядом с найденным постом — URL оффера автоматически добавится в твой реестр офферов.

## 7. Прогрев аккаунтов `/warmup`

Новые аккаунты нельзя сразу использовать для рассылок — Telegram заблокирует.
Прогрев имитирует живого человека в течение 5 дней.

### Как прогревать
1. Зайди в Прогрев
2. Нажми ⚙ рядом с аккаунтом → выбери **нишу** (крипта, гемблинг и т.д.)
   Ниша определяет на какие каналы подписывается аккаунт
3. Нажми **"Прогреть всех"**
4. Система каждые 2 часа делает шаг прогрева автоматически

### Что делает прогрев каждый день
- День 1: Читает каналы, подписывается на 3 канала по нише
- День 2: Читает, ставит 1-3 реакции, подписывается ещё
- День 3: Читает, ставит реакции, начинает писать в чаты
- День 4-5: Более активное поведение

### Детектор бана
Каждые 3 дня система проверяет аккаунт через @SpamBot.
Если аккаунт забанен — автоматически заменяется в активных кампаниях.

### Статусы
- **Офлайн** — не подключён
- **Прогрев** — идёт прогрев (1-4 день)
- **Активен** — готов к рассылкам (5+ день)
- **Лимит** — получил временное ограничение
- **Бан** — заблокирован Telegram

## 8. Воронки `/funnels`

Бот-преленд: пользователь пишет `/start` боту → проходит цепочку вопросов → получает оффер.

### Создать воронку
1. Нажми **"+ Создать воронку"**
2. Или выбери готовый пресет: **"Крипто оффер"** или **"Беттинг воронка"**
3. В конструкторе добавляй шаги:
   - **Сообщение** — бот просто отправляет текст
   - **Вопрос** — кнопки с вариантами ответа
   - **Редирект** — кнопка-ссылка на оффер
4. Для каждого шага укажи задержку в секундах

### Как работает
Пользователь пишет `/start` → бот ведёт по шагам → в конце отправляет ссылку на оффер.

### Статистика воронки
- **Старты** — сколько людей начали
- **Завершено** — сколько дошли до конца

## 9. Трекер `/tracker`

Отслеживает клики и конверсии.

### Создать UTM-ссылку
1. Трекер → **"+ Создать ссылку"**
2. Введи URL оффера
3. Получи короткую ссылку вида: `passionate-courtesy-production.up.railway.app/r/abc123`

### Постбэк для CPA-сетей
Вставь этот URL в настройки CPA-сети:
```
https://passionate-courtesy-production.up.railway.app/api/postback?sub_id={sub_id}&event=lead&payout={payout}
```
Когда придёт конверсия — система автоматически её запишет и пришлёт уведомление в Telegram.

### Уведомления
Чтобы получать уведомления в Telegram:
1. Напиши @userinfobot в Telegram
2. Скопируй свой ID
3. В Railway → сайт → Variables → добавь `ADMIN_CHAT_ID=твой_id`

## 10. ROI Калькулятор `/roi`

Считает прибыльность кампании ДО запуска.

### Как использовать
Двигай слайдеры:
- **Аккаунты/Прокси/Сервер** — твои расходы
- **Сообщений** — сколько планируешь отправить
- **Доставляемость** — % сообщений которые дойдут (обычно 70-85%)
- **CTR** — % людей которые перейдут по ссылке (обычно 3-8%)
- **CR** — % людей которые совершат целевое действие (обычно 5-15%)
- **Выплата за лид** — сколько платит CPA-сеть

### Что показывает
- Прибыль/убыток
- ROI в процентах
- EPC (доход на клик)
- CPL (стоимость лида)
- Сколько лидов нужно для окупаемости

## Типичный рабочий день арбитражника

1. **Утро** — проверить Обзор (новые конверсии, статус аккаунтов)
2. **Добавить новые каналы** через Поиск → запустить парсинг
3. **ROI Калькулятор** — просчитать новую кампанию
4. **Запустить рассылку** с новым текстом
5. **Шпион** — посмотреть что рекламируют конкуренты, найти новые офферы
6. **Вечер** — проверить аналитику, посмотреть что сработало

## Частые вопросы

**Аккаунт получил статус "Бан" — что делать?**
Система автоматически заменит его в активных кампаниях. Просто добавь новый аккаунт и запусти прогрев.

**Рассылка зависла в статусе "Идёт" — почему?**
Проверь что воркер запущен (Railway → Воркер → View logs). Если воркер крашнулся — перезапусти его.

**Сколько аккаунтов нужно для нормальной работы?**
Минимум 3-5 для начала. Оптимально 10-20 с ротацией.

**Какой лимит сообщений ставить?**
Для новых аккаунтов: 20-30/день. Для прогретых: 40-80/день. Не больше.
